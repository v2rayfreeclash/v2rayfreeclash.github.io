一、工具定位：不只是VPN的代理专家

Shadowrocket诞生于iOS生态，专为追求网络自由的技术爱好者设计。与传统VPN应用不同，它采用模块化架构，支持Shadowsocks、Vmess、Trojan等主流代理协议，甚至能通过自定义配置兼容新兴技术。这种"协议中立"的特性，使其能适应不断演变的网络封锁技术——当某类协议被识别封锁时，用户可快速切换其他通道，保持连接韧性。

更值得称道的是其流量分治能力。通过规则引擎，用户可实现"国内直连、国外代理"的智能分流，避免全局代理带来的延迟损耗。例如设定微信、支付宝等高频应用直连，而Google、Twitter等走代理通道，既保障速度又突破封锁。这种颗粒度控制，远超大多数只能简单开关的VPN应用。

二、核心功能全景解析

1. 多协议支持：应对复杂网络环境的武器库

  • Shadowsocks：轻量级加密代理的标杆，适合移动端低功耗需求
  • Vmess（V2Ray核心协议）：支持动态端口和元数据混淆，对抗深度包检测
  • Trojan：伪装成HTTPS流量，在严格审查网络中隐蔽穿行
  • 原生支持SOCKS5/HTTP等传统代理，兼容企业级网络环境

2. 规则引擎：网络流量的智能调度中心

用户可基于域名、IP、地域等多维度制定规则：
```

示例规则逻辑

DOMAIN-SUFFIX,wechat.com,DIRECT // 微信直连
DOMAIN-KEYWORD,google,PROXY // Google相关走代理
GEOIP,CN,DIRECT // 中国IP直连
FINAL,PROXY // 其余流量默认代理
```
这种类编程的规则体系，配合定期更新的规则订阅功能，让网络管理如同配置防火墙般精准。

3. 流量可视化：数据使用的透明化管理

实时显示上下行流量曲线，支持分应用统计。某用户曾通过此功能发现某新闻APP在后台持续跨国传输数据，及时切断了潜在隐私泄露风险。这种透明化设计，在"黑箱操作"泛滥的当下尤为珍贵。

1. 获取服务器信息（地址、端口、密码、TLS指纹等）
